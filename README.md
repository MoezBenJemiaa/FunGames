## Getting Started

### Prerequisites

- [Git](https://git-scm.com/)
- [Node.js](https://nodejs.org/) (if games are web-based)
- Any other dependencies specific to games (see individual game folders)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/RostomTurki/FunGames.git
   cd FunGames
   ```

2. **Install dependencies**
   - For web-based games, run:
     ```bash
     npm install
     ```
   - For other languages, see instructions in the respective game folders.

3. **Run a game**
   - Each game has its own README/instructions inside its folder.
